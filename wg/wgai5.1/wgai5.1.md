## 🔌 API对接示例

### 快速接入只需 3 步

```bash
# 1. 启动服务
docker run -d -p 9999:9999 wgai/platform:v5.1

# 2. 调用识别接口（Python示例）
import requests

url = "http://localhost:9999/api/face/recognize"
files = {'image': open('photo.jpg', 'rb')}
response = requests.post(url, files=files)

# 3. 获取识别结果
result = response.json()
print(f"识别结果: {result['name']}, 置信度: {result['confidence']}")
```

## 🎯 快速开始

### 方式一：Docker 一键部署（推荐）
```bash
# 拉取最新镜像
docker pull wgai/platform:v5.1

# 启动服务
docker run -d \
  -p 9999:9999 \
  -v /data/wgai:/app/data \
  --name wgai-platform \
  wgai/platform:v5.1

# 访问管理后台
# 浏览器打开：http://localhost:9999
```

### 方式二：安装包部署
```bash
# 1. 下载安装包
wget https://gitee.com/dromara/wgai/releases/v5.1/wgai-v5.1.zip

# 2. 解压并启动
unzip wgai-v5.1.zip
cd wgai-v5.1
./start.sh

# 3. 访问系统
# 浏览器打开：http://localhost:9999
```

### 方式三：源码编译
```bash
# 1. 克隆代码
git clone https://gitee.com/dromara/wgai.git

# 2. 编译运行
cd wgai
mvn clean package
java -jar target/wgai.jar
```
